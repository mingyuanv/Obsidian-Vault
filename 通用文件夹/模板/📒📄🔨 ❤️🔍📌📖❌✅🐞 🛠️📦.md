# 其他
|类型|含义|图文字|说明|
|---|---|---|---|
|⚠️|**警告**|`⚠️` (Warning)|一般提示潜在风险或注意事项|
|❗|**错误**|`❗` (Error)|用于明确的错误、失败或异常|
|❌|**失败**|`❌` (Cross Mark)|表示操作失败或不允许|
|🚫|**禁止**|`🚫` (Prohibited)|表示禁止执行或非法操作|
|🛑|**严重错误**|`🛑` (Stop Sign)|表示严重错误或需要立刻停止|
|⛔|**阻止**|`⛔` (No Entry)|表示拒绝访问或阻塞|
|⚡|**异常**|`⚡` (High Voltage)|可用于表示崩溃、闪退、断电类异常|
|💥|**崩溃**|`💥` (Collision)|用于严重崩溃、系统故障场景|
|🐞|**Bug**|`🐞` (Bug)|表示发现软件缺陷或调试输出|
|🧨|**炸了**|`🧨` (Explosion)|轻松幽默地表达“挂了”、“炸了”|
