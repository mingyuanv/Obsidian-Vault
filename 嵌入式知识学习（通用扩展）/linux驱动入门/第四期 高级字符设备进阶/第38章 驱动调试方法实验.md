# 一、驱动调试方法

## 方法1：dump_stack函数
### 1 、打印内核调用堆栈，并打印函数的调用关系。


### 2 、实验：以最简单的helloworld驱动为例进行dump_stack函数演示
```c
#include <linux/init.h>
#include <linux/module.h>
#include <linux/kernel.h> 

static int __init timer_mod_init(void){

    printk(KERN_EMERG "hello_word_init");
    dump_stack();

    return 0;
}

static void __exit timer_mod_exit(void)
{
    printk("module exit \n");

}

module_init(timer_mod_init);
module_exit(timer_mod_exit);

MODULE_LICENSE("GPL v2");
MODULE_AUTHOR("topeet");
```

##  方法2：WARN_ON(condition)函数
### 1 、在括号中的条件成立时，内核会抛出栈回溯，打印函数的调用关系。

> WARN_ON**实际上也是调用dump_stack**，只是多了参数condition判断条件是否成立，例如WARN_ON (1)则条件判断成功，函数会成功执行。


### 2 、通常用于内核抛出一个警告，暗示某种不太合理的事情发生了。


### 3 、实验：以最简单的helloworld驱动为例进行WARN_ON函数演示
```c
#include <linux/init.h>
#include <linux/module.h>
#include <linux/kernel.h> 

static int __init timer_mod_init(void){

    printk(KERN_EMERG "hello_word_init");
    //dump_stack();
    WARN_ON(1);

    return 0;
}

static void __exit timer_mod_exit(void)
{
    printk("module exit \n");
}

module_init(timer_mod_init);
module_exit(timer_mod_exit);

MODULE_LICENSE("GPL v2");
MODULE_AUTHOR("topeet");

```

## 方法3：BUG_ON (condition)函数（报错）
### 1 、BUG_ON (condition)函数作用
> 它非常像一个内核运行时的断言，意味着本来不该执行到BUG_ON()这条语句，一旦**BUG_ON()执行内核就会立刻抛出oops，导致栈的回溯和错误信息的打印**。
> 大部分体系结构把BUG()和BUG_ON()定义成某种**非法操作**，这样自然会产生需要的oops。参数condition判断条件是否成立，例如BUG_ON (1)则**条件判断成功，函数会成功执行。**


### 2 、实验代码
```c
#include <linux/init.h>
#include <linux/module.h>
#include <linux/kernel.h> 

static int __init timer_mod_init(void){

    printk(KERN_EMERG "hello_word_init");
    //dump_stack();
    //WARN_ON(1);
    BUG_ON(1);
    return 0;
}

static void __exit timer_mod_exit(void)
{
    printk("module exit \n");
}

module_init(timer_mod_init);
module_exit(timer_mod_exit);

MODULE_LICENSE("GPL v2");
MODULE_AUTHOR("topeet");
```

## 方法4：panic (fmt…)函数（死机）
### 1 、输出打印会造成系统死机并将函数的调用关系以及寄存器值就都打印了出来。


### 2 、实验代码
```c
#include <linux/init.h>
#include <linux/module.h>
#include <linux/kernel.h> 



static int __init timer_mod_init(void){

    printk(KERN_EMERG "hello_word_init");
    //dump_stack();
    //WARN_ON(1);
    //BUG_ON(1);
    panic("!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!");

    return 0;
}


static void __exit timer_mod_exit(void)
{


    printk("module exit \n");

}

module_init(timer_mod_init);
module_exit(timer_mod_exit);

MODULE_LICENSE("GPL v2");
MODULE_AUTHOR("topeet");

```
